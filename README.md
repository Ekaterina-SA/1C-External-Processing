# 1C-External-Processing
Внешняя обработка разработана в рамках технического задания. Реализована для работы с ведомостями и их табличными частями (договорами). Обработка содержит две команды: «Заполнить нужное» «Удалить лишнее».
Функционал «Заполнить нужное» - 
Обработка:
Программно формирует таблицу ведомостей (ссылки на документы).
Отбирает ведомости по условию:
Дата начала действия первого активного договора
равна / не равна, минимальной / максимальной дате начала действия договора в табличной части.
Выполняет динамическое исключение ведомостей в зависимости от текущей даты: 11–20 число — исключаются документы с чётным числом дня.
Сортирует ведомости по возрастанию даты документа.
Выводит на форму только колонку со ссылками.
Выделяет светло-зелёным цветом строки, где последний статус равен: «Акцептован» (по данным регистра сведений «ИсторияСтатусовДокументов»).
Условное оформление:
Для женщин — реализовано через конструктор формы.
Функционал «Удалить лишнее» - 
Удаляет строки, где количество договоров в табличной части чётное.
Выводит сообщение пользователю в формате: Удалено документов: N (№1, №2, №3), сортировка номеров — по возрастанию.
Реализация информирования пользователя: в зависимости от длины фамилии используются различные механизмы: 10–11 — элемент основной формы.
Платформа:1С:Предприятие 8.3
Особенности реализации: Использованы серверные и клиентские процедуры, оптимизированы обращения к регистрам, код снабжён комментариями
